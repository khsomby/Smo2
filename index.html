Facebook Messenger Bot by Somby Ny Aina

This project is a Facebook Messenger bot built using Node.js and Express. It integrates with an AI service to provide responses to user messages.

Key features:

  • Handles incoming messages and responds based on AI-generated content
  • Integrates with Facebook's Messenger Platform API
  • Built with Express.js and uses Axios for HTTP requests
